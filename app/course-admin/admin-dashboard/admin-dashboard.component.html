Bienvenido al administrador de curso :)

En esta sección podrás modificar todo el contenido del curso, agregar nuevos alumnos, crear grupos y más. Te dejamos una explicación de cada una de las secciones (a las cuales podes acceder con los botones de arriba)

  • Inicio: ¡Lo que estás leyendo en este momento! Por ahora solo tendrá este tutorial
  • Contenido: Aquí podrás crear contenido para que tus alumnos vean. Podés hacer foros, evaluaciones, solicitar entregas, subir archivos para que ellos descarguen y más. El contenido está dividio en capítulos, y cada capítulo está divido en tabs (pestañas) para que puedas organizar el material como prefieras ¡No es necesario que crées muchas tabs ni muchos capítulos si no querés!
  • Videoconferencia: Entrando aquí podés comenzar una videoconferencia a la que tendrán acceso todos los estudiantes de este curso.
  • Alumnos: En esta pestaña podrás ver todos los alumnos de este curso, y brindarle acceso a otros usuarios que estén registrados en la plataforma. Además, si no sos el unico profesor del curso podés darle permiso de administrador a otro usuario para que tenga acceso a este panel
  • Entregas: Todas las evaluaciones creadas en la sección Editar contenido del curso podrás verlas aqui, y con ellas las respuestas de tus alumnos.

¡Esperamos que este resumen te ayude a utilizar la plataforma! Si tenés alguna pregunta no dudes en escribirnos a nuestras redes sociales (las podés encontrar mas abajo) y estaremos felices de ayudarte.
